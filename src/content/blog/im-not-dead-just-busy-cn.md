好久没更新博客了，先说声抱歉。

我不是死了，我是最近忙死了。

## 最近在忙什么

抱歉朋友们，最近我真是忙过头了，一个是准备实习的事情，经过很长的流程也是得到一研究院的实习。另一个就是走前突然有人找上我能不能帮他们写一个平台，一开始只是帮学生打比赛，但是后面发现能不能卖给他们学校的学院就疯狂在干在更新。

不过这个项目还是很有意思的，我可以透露一下目前的技术栈，我昨天刚整了一个Linux服务器然后部署完Gitea，建了一个代码仓库把最近的进度传上去然后把项目服务部署到服务器上上线，做的也是相当的累，我其实是第一次做这种比较完善的Linux提供网页服务，不过由于这个项目保密性要求，我无法将源码和一些东西开源，不过我可以通过这次经历写好几篇相关的博客。因为这是一个很大的要给学校用的系统。

| 层级     | 技术                                | 说明                                                         |
| -------- | ----------------------------------- | ------------------------------------------------------------ |
| 前端框架 | Vue 3                               | 使用组合式 API，页面以单文件组件为主                         |
| 前端构建 | Vite                                | 本地开发与生产构建入口                                       |
| 前端样式 | Tailwind CSS                        | 官网与后台页面的主要样式系统                                 |
| 前端路由 | Vue Router                          | 官网、登录页、后台页、可视化设计页路由                       |
| 前端图标 | lucide-vue-next                     | 用于后台、登录页和展示页面图标                               |
| 登录服务 | Node.js + Express                   | 提供 `/api/auth/login`、`/api/auth/me` 等鉴权接口            |
| 鉴权方式 | JWT                                 | 登录后由前端保存 token，用于访问受保护页面                   |
| 数据库   | MySQL                               | 存储三端用户账号，初始化 SQL 位于 `back_end/sql/`            |
| 分析服务 | Python + FastAPI                    | 提供酒店评论爬取、评论数据读取、情感分析任务接口             |
| 情感分析 | Transformer 模型预留 / 本地模型文件 | 模型目录位于 `back_end/models/`，后续继续完善推理链路        |
| 进程管理 | pm2                                 | Linux 服务器上管理 `hotel-auth` 与 `hotel-analysis` 服务     |
| Web 入口 | nginx                               | 托管前端静态资源，并代理 `/api` 与 `/analysis`               |
| 自动部署 | Gitea Webhook + systemd + bash      | push 到 `main` 后触发本机 Webhook，统一执行 `scripts/deploy.sh` |
| 部署系统 | Ubuntu Linux                        | 当前服务器部署目标为 Ubuntu/Debian 系 Linux                  |

我也快去实习了，也最近在忙别的项目，真的太忙了平时写不动，其实我本来想写一些关于Matlab的教程的，例如Mathworks官方的Matlab MCP如何配置效果如何，如何利用这个东西设计一个完善的光伏停车场的EMS系统之类。但是结果我才研究到一半，接上这个新活天天疯狂写疯狂测我真的是没有什么精力了。
